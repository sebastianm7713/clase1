1. HTML

HOME CSS JAVASCRIPT

HTML (lenguaje de marcado de hipertexto) es unlenguaje de marcado de hipertexto que se utiliza para estructurar y dar forma a las páginas web y su contenido. Se trata de un código que permite organizar textos, imágenes, vídeos, botones y otros elementos de navegación. HTML es la base de la World Wide Web y se utiliza en el 94% de los sitios web. Es un conocimiento indispensable para quienes se dedican a la creación de páginas web, ya sea de manera profesional o como hobby.

CSS (Cascading Style Sheets) es un lenguaje de programación que se utiliza para dar formato a los elementos de un sitio web: Define los colores, fuentes, tamaños, posicionamiento y otros valores estéticos Se encarga de presentar visualmente los datos de un sitio web a través de un navegador




3. JAVASCRIPT

HOME HTML CSS

JavaScript es un lenguaje de programación que los desarrolladores utilizan para hacer páginas web interactivas. Desde actualizar fuentes de redes sociales a mostrar animaciones y mapas interactivos, las funciones de JavaScript pueden mejorar la experiencia del usuario de un sitio web.
